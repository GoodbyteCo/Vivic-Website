# Vivic Research

Vivic Research website built in [Astro](https://astro.build).

## Operational Notes

### Local Development

Install dependencies:

```bash
npm install
```

| COMMAND        | DESCRIPTION                             |
|:---------------|:----------------------------------------|
| `npm run dev`  | Run the site locally                    |
| `npm run lint` | Check for formatting errors             |

### Deployment

Deploys automatically via Netlify.
