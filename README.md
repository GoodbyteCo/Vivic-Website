# Vivic Research

Vivic Research site built in [Astro](https://astro.build)

## Operational Notes

### Local Development

Install dependencies:

```bash
npm install
```

| COMMAND       | DESCRIPTION                             |
|:--------------|:----------------------------------------|
| `npm run dev` | Run the site locally                    |
